## Resumo | 3
- Modelos de Serviços
### IaaS,PaaS,SaaS

- IaaS : Infraestrutura como serviço, mais nossas (Dev) responsabilidade.

- PaaS : Plataforma como serviço, ambas responsabilidade. 

- SaaS : Software como serviço. menos responsabilidade.
